## To run locally

- Create a virtual environment:

  ```bash
  python3 -m venv venv
  ```

- Activate the virtual environment:

  ```bash
  source venv/bin/activate
  ```

- Install dependencies:

  ```bash
  pip install -r requirements.txt
  pip install -r requirements-dev.txt
  ```

- Run the app:

  ```bash
  flask run
  ```
